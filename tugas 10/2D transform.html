CSS 2D Transform

Koleksi contoh transformasi 2D CSS yang interaktif dan menarik

Translate

Memindahkan elemen dari posisi aslinya

X
Hover untuk melihat efek translateX(50px)

Scale

Mengubah ukuran elemen

S
Hover untuk melihat efek scale(1.3)

Rotate

Memutar elemen dengan sudut tertentu

R
Hover untuk melihat efek rotate(45deg)

Skew

Memiringkan elemen pada sumbu tertentu

K
Hover untuk melihat efek skewX(20deg)

Multiple

Kombinasi beberapa transformasi

M
Hover untuk kombinasi translate, scale, rotate

Matrix

Transformasi menggunakan matrix 2D

T
Hover untuk melihat efek matrix transform
